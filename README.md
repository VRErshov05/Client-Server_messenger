# Курсовая работа на C++

## Client - Server - Client(s) "messenger"

### Задание курсовой работы:

1. Клиент может отправить конкретному клиенту сообщение (через сервер). Если второй клиент онлайн, то должен получить сообщение. Если второй клиент не онлайн, возвращаем ошибку первому клиенту.
2. Клиент может всем одновременно отправить сообщение.
3. Клиент может запросить содержимое файла с сервера. Если файла не существует, возвращаем ошибку клиенту.
4. Клиент может отправлять файлы на сервер.
